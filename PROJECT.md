## 3. Технологии

- Чистый HTML + CSS + JS в одном файле `index.html` (без сборки, без фреймворков).

## 4. Хранение данных (ключи в localStorage + Firebase /app)

Слой `ld(key, default)` / `sv(key, value)` читает из памяти `MEM` (зеркало localStorage),
пишет в localStorage + Firebase. `window.__applyRemote` принимает обновления из облака.

| Ключ | Содержимое |
|---|---|
| `c_ГГГГ_М_Д` | отметки дня: `{d:bool, o:bool}` (d=Дима/зарядка, o=Оля/таблетки). Месяц 0–11. |
| `ach` | объект `{achId: true}` — выполненные ачивки |
| `customAchs` | массив пользовательских ачивок `{id:'c..', title, cat, svg}` |
| `together_since` | дата начала отношений (строка `YYYY-MM-DD`) |
| `events` | массив важных дат `{id, title, date}` (будущие + прошедшие) |
| `letters` | письма-капсулы `{id, from, unlock, text, cover, created}` |
| `photos` | фото `{id, data(base64 jpeg ~1000px), by, cap, ts}` |
| `tracks` | музыка `{id, url, title, by, ts}` |
| `questions` | вопрос дня: `{dateKey:{d:'',o:''}}`, dateKey=`Y-M-D` |
| `q_who` | (устар.) кто отвечает — заменён на `me` |
| `me` | кто вошёл: 'd'/'o' (ЛОКАЛЬНО). Подставляется во все «кто добавил» |
| `pet` | питомец Амням: `{xp, claimed:{ключи событий}}`. 35 уровней эволюции (генератор `window.AMNYAM`) |
| `tierlists` | тирлисты: `{weeks:{weekKey:{themeId,labels[],place:{d:{},o:{}}}}, votes:{weekKey:{cands[],d:{},o:{}}}}`. weekKey=понедельник `Y-M-D`, place — `{tN:tierIdx}` по индексу элемента |
| `lastBackup` | время последнего бэкапа (ЛОКАЛЬНО, напоминание только Диме) |
| `muted` | звук выкл (`'1'`) — ЛОКАЛЬНО на устройстве, не синхронизируется |

## 5. Функциональность (вкладки)

1. **Календарь** — две привычки (Дима: зарядка, Оля: таблетки), отметки по дням, стрики.
   - Стрик: считается по вчерашний день; сегодня необязательно; обнуляется только на след. день после пропуска.
2. **Вместе** — живой счётчик дней/часов/минут/секунд с даты `together_since`; вехи (недели, годы, до 100 дней, годовщина); раздел «Важные даты» (будущие «через N дней» и прошедшие «N дней назад»).
3. **Письма** — капсулы времени: текст + дата открытия + обложка (5 узорных стилей). До даты запечатано.
4. **Фото** — загрузка с авто-сжатием на клиенте (~1000px JPEG), подпись, автор, лайтбокс.
5. **Музыка** — плейлист; YouTube → аудио мини-плеер (скрытое видео, low quality, экономия трафика); прочие ссылки — кнопкой.
6. **Ачивки** — 2 раздела («Поцелуи» / «Остальное»), прогресс-бары, золотые SVG-иконки в неоновых HUD-рамках, добавление своих, авто-подгон размера текста.
7. **Статистика** — стрики, выполнение за 30 дней, кол-во ачивок, бары.
8. **Тирлисты** — еженедельный тирлист (тиры S/A/B/C/D, названия редактируются). Раскладка тапом (элемент → тир). Дима и Оля раскладывают независимо; раскладка второго видна сразу (только просмотр), без обязательства заполнить свою. За готовый тирлист +30 XP питомцу. Голосование за тему следующей недели (5 случайных тем, галочки от обоих): в понедельник авто-выбор случайной из отмеченных обоими, иначе из отмеченных хоть кем-то, иначе новая случайная. Выбор детерминированный (сид по неделе → совпадает на обоих устройствах). 40 тем (~16–24 элемента); первая тема — Фрукты. Архив прошлых недель (обе раскладки рядом).

Прочее: звук набора + бип доступа (кнопка 🔊/🔇 в шапке), конфетти, дешифровка цифр при переходе вкладок.

## 6. Дизайн (аниме-киберпанк)

- Палитра: void `#06070d`, циан `#00f0ff`, магента `#ff2a6d` (Дима), зелёный `#05ffa1` (Оля).
- Эффекты: неоновая сетка-перспектива, сканлайны+развёртка, виньетка, дата-дождь (canvas), неоновый курсор-трейл, глитч (логотип + ховер карточек), HUD-углы (clip-path), свечение.
- Вход = «экран взлома»: печатающийся boot-лог + ACCESS GRANTED/DENIED.
- **Двухшаговый вход** (защита от случайного входа под чужим профилем):
  - Шаг 1 — ввод пароля (`#login-step-pwd`), при верном → вспышка «КОД ПРИНЯТ».
  - Шаг 2 — полноэкранный киберпанк-сплит выбора (`#login-step-who`): две половины с неоновыми фонами (Дима — голубой/электрик + техно-сетка; Оля — магента/фиолет ауры), крупные имена шрифтом Tektur, светящийся шов. Наведение раздвигает половину и затемняет вторую (`войти ▸`), клик = вход → «ДОСТУП РАЗРЕШЁН» и установка `me`.
  - Функции: `login()` (только проверка пароля), `enterAs('d'/'o')` (вход + `me`), `logout()` сбрасывает на шаг пароля.
  - Мобильно: `@media(max-width:640px)` — вертикальный стек, текст `16vw`, подсказка/неон видны сразу; `@media(hover:none)` — касание подсвечивает половину вместо hover.
- Стиль реализован как слой-надстройка `<style>` + 2 `<script>` в конце файла (после основного кода) — легко откатывается.
- Базовый «тёплый кремово-золотой» дизайн всё ещё под низом (первый `<style>`), киберпанк его перекрывает через `!important`.

Теперь индекс по локальной календарной дате через `Date.UTC(y,m,d)`, одинаково для текущего дня и архива — коммит `b003e04`
- **«Вопрос дня» — ответ второго снова скрыт** до твоего ответа (откат «видны сразу» только для вопроса; архив открыт, тирлисты не трогали). В `renderQuestion` блок партнёра показывает «🔒 ответ скрыт», пока `myAns` пуст — коммит `47e8c5c`

Тиры: Зерно→Малыш→Юный→Кибер-форма→Усиленный→Продвинутый→Легенда. `PET_MAXLVL` 18→35. Кривая XP смягчена: `lvlThreshold = Σ 45·1.13^(i-1)` (раньше `80·1.45^(i-1)`, при которой 35-й был недостижим); теперь ур.35 ≈ 21 700 XP.
